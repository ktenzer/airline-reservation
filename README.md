# airline-reservation
Airline Reservation System Built on Temporal

# Setup
```bash
$ mkdir .venv
$ curl -sSL https://install.python-poetry.org | python3 -
$ poetry install
```

# Run App
```bash
$ poetry run python app.py
```

# Run Worker
```bash
$ poetry run python worker.py
```
